# Tom's web toolbox

A Quasar-based project featuring various tools useful for its developer @tcalmant.

Project released under the terms of the [Apache 2.0 licence](./LICENSE).

## Command lines

- Install dependencies
  ```bash
  npm ci
  ```
- Start the dev server
  ```bash
  quasar dev
  ```
- Linting
  ```bash
  npm run lint
  ```
- Building
  ```bash
  quasar build
  ```
